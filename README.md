## 📌 **Important: LED SVG Rendering Instructions**

> **If the LED typing board above doesn't render properly:**
> 1. The SVG uses `readme-typing-svg.demolab.com` (alternative: `readme-typing-svg.herokuapp.com`)
> 2. LED-style fonts: `Orbitron`, `Share Tech Mono`, `Courier New`
> 3. For custom styling: Edit `font=`, `color=`, `background=`, and `lines=` parameters in the URL
> 4. Yellow (`FFD700`) on black (`000000`) creates authentic LED bus sign look
> 5. Add more scrolling text by adding `+` symbols between words in `lines=` parameter
